# I/O

## Output

```axis
write("Hello ")       # no newline
writeln("World!")      # with newline
writeln(42)            # works with numbers
writeln(True)          # works with booleans
```

## Input

```axis
name: str = readln()       # read a line of text
ch: str = readchar()       # read a single character
num: i32 = read()          # read an integer
```

`read()` reads an integer from stdin. If the input is not a valid integer, the `read_failed` flag is set (see below).

## Read Error Handling

```axis
x: i32 = read()
when read_failed:
    writeln("That wasn't a number")
```

`read_failed` is a built-in boolean that is `True` if the last `read()` call failed to parse an integer.
